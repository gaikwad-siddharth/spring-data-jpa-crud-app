# 📦 Spring Data JPA CRUD Application

A Spring Boot application demonstrating **CRUD (Create, Read, Update, Delete)** operations using **Spring Data JPA** and the **Repository pattern**.

---

## 📌 Project Overview

This project showcases how to build a backend application that interacts with a database using Spring Boot and JPA. It follows a clean layered architecture and demonstrates real-world backend development practices.

---

## 🧠 Concepts Covered

- Spring Boot
- Spring Data JPA
- CRUD Operations
- Repository Pattern
- Entity Mapping (JPA/Hibernate)
- Layered Architecture (Controller → Service → Repository)
- Database Integration

## 🔄 CRUD Operations

| Operation | Description |
|----------|------------|
| Create   | Add new record to database |
| Read     | Fetch records from database |
| Update   | Modify existing data |
| Delete   | Remove records |

---

## ▶️ How to Run

1. Clone the repository
2. Open in IDE (Eclipse/IntelliJ)
3. Configure database in `application.properties`
4. Run the main class
